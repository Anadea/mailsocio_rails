# Mailsocio Rails

ActionMailer plugin to use mailsocio to deliver emails

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'mailsocio_rails', github: 'Anadea/mailsocio_rails'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install mailsocio_rails
